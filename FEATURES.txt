ExcelAI v2.0 — Feature Catalog
==============================

CORE ENGINE
-----------
- AI-powered Excel automation via Ollama (local LLM inference)
- Live Excel control via xlwings (read, write, format, chart, formula)
- Code validation (AST-based safety checks before execution)
- Dry-run preview (analyze what code will do without executing)
- Analysis mode (AI answers data questions instead of writing code)
- Auto-run mode (execute AI-generated code automatically)
- Session persistence (save/load model, settings, conversation)
- Undo/Redo support for Excel operations
- Batch scheduler for automated jobs
- VBA to Python conversion

IMAGE / OCR PIPELINE
--------------------
- Image preprocessing (binarization, contrast, deskew, denoise)
- Multi-pass OCR extraction (structure detection + data extraction)
- Two-page spread support (left + right halves)
- Post-OCR validation (IFSC, account numbers, bank names, serials)
- Handwriting disambiguation rules

SAFETY & VALIDATION
--------------------
- AST-based code validation (blocks os, subprocess, eval, open, etc.)
- Blocked dunder attribute access (__class__, __globals__, etc.)
- Blocked dangerous imports (27 modules)
- Dry-run analysis (preview cells written, formulas, formatting)
- Undo manager for reverting changes
